本作品为2022中国大学生计算机设计大赛——AI实践赛参赛作品，最终在比赛中获得国家三等奖。

本作品通过设计和研发网上预约挂号聊天机器人，充分发挥计算机强大的信息检索与处理能力，打破了各医院预约挂号系统间的壁垒。将所有可查询到的医院信息集成于一体，为用户查询各医院的号源信息，提供更加便捷的“一站式”挂号信息查询服务。
